# ✈️ Airline Passenger Satisfaction Prediction

## 📌 Project Overview

This project focuses on predicting airline passenger satisfaction using Machine Learning techniques. By analyzing passenger travel experiences and service-related factors, the model classifies whether a customer is satisfied or dissatisfied with the airline service.

The goal is to help airlines understand the key drivers of customer satisfaction and make data-driven decisions to improve passenger experience.

---

## 🎯 Objectives

* Analyze factors influencing passenger satisfaction.
* Perform data preprocessing and feature engineering.
* Build and evaluate machine learning classification models.
* Identify the most important features affecting customer satisfaction.
* Predict passenger satisfaction with high accuracy.

---

## 📊 Dataset Features

The dataset contains information related to:

* Customer Type
* Age
* Type of Travel
* Class
* Flight Distance
* Departure Delay
* Arrival Delay
* Inflight Wi-Fi Service
* Online Booking Experience
* Seat Comfort
* Food & Drink
* Online Boarding
* Inflight Entertainment
* Baggage Handling
* Check-in Service
* Cleanliness
* And other service-related attributes

**Target Variable:** Satisfaction (Satisfied / Neutral or Dissatisfied)

## 🤖 Machine Learning Models

The following classification models were evaluated:

* Logistic Regression
* Random Forest Classifier

The best-performing model was selected based on evaluation metrics.

---

## 📈 Model Evaluation

Performance was assessed using:

* Accuracy Score
* Precision
* Recall
* F1-Score
* Confusion Matrix

---

## 🚀 Deployment

The trained model was deployed using **FastAPI**, enabling users to provide passenger details and receive real-time satisfaction predictions through an API endpoint.

---


## 📌 Key Insights

* Online Boarding and Inflight Entertainment significantly impact customer satisfaction.
* Departure Delay and Arrival Delay are strongly correlated.
* Business Class passengers generally report higher satisfaction levels.
* Service quality features contribute more to satisfaction prediction than demographic factors.
